# Contributing to USMC

Thanks for your interest in contributing!

## Getting Started

1. Fork and clone: `git clone https://github.com/YOUR_USERNAME/usmc.git`
2. Install: `pip install -e .`
3. No external dependencies needed (stdlib only)

## How to Contribute

- **Bug fixes**: Open an Issue or submit a PR directly
- **New features**: Open an Issue first to discuss
- **New memory backends**: Very welcome as plugins
- **Documentation**: Always appreciated

### Good First Issues

Look for [`good first issue`](https://github.com/ellmos-ai/usmc/labels/good%20first%20issue) labels.

## Guidelines

- Keep zero-dependency philosophy (stdlib only)
- Python 3.10+, type hints where practical
- One feature/fix per PR
- Update docs if behavior changes
